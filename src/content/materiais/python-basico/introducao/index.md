# Introdução à Programação

Bem-vindo ao mundo da programação! Este material vai te guiar pelos primeiros passos nessa jornada incrível.

## O que é Programação?

Programação é a arte de criar instruções para que computadores executem tarefas específicas. É como escrever uma receita de bolo, mas para máquinas!

> "Programar é ensinar uma máquina a pensar." - Anônimo

### Por que aprender a programar?

- **Resolução de problemas**: Desenvolve seu raciocínio lógico
- **Criatividade**: Permite transformar ideias em realidade
- **Mercado de trabalho**: Habilidade cada vez mais valorizada
- **Automação**: Automatize tarefas repetitivas do dia a dia

## Conceitos Fundamentais

### Algoritmos

Um algoritmo é uma sequência de passos bem definidos para resolver um problema. Pense em como você faria para preparar um café:

1. Encher a chaleira com água
2. Ligar a chaleira
3. Aguardar a água ferver
4. Colocar café no filtro
5. Despejar a água quente sobre o café
6. Aguardar o café coar
7. Servir o café

Isso é um algoritmo! Na programação, escrevemos algoritmos usando uma **linguagem de programação**.

### Variáveis

Variáveis são como "caixas" onde guardamos informações. Por exemplo:

```python
nome = "Maria"
idade = 21
altura = 1.65
```

Existem diferentes tipos de variáveis:

| Tipo | Descrição | Exemplo |
|------|-----------|---------|
| String | Texto | `"Olá, mundo!"` |
| Integer | Número inteiro | `42` |
| Float | Número decimal | `3.14` |
| Boolean | Verdadeiro/Falso | `True` |

### Estruturas de Controle

#### Condicionais (if/else)

Permitem que o programa tome decisões:

```python
idade = 18

if idade >= 18:
    print("Você é maior de idade")
else:
    print("Você é menor de idade")
```

#### Loops (repetições)

Permitem repetir ações:

```python
# Loop for
for i in range(5):
    print(f"Contagem: {i}")

# Loop while
contador = 0
while contador < 5:
    print(f"Valor: {contador}")
    contador += 1
```

## Sua Primeira Linguagem: Python
